# Scratchp
-A játékban cicák esnek le az égből, amelyeket a játékosnak egy kosár segítségével kell elkapnia. A kosarat a nyilakkal lehet mozgatni. Minden elkapott cica növeli a pontszámot. A játékos három élettel (szívvel) rendelkezik. Ha egy cica leesik a földre és nem sikerül elkapni, a játékos elveszít egy életet. Ha az összes élet elfogy, a játék véget ér.
